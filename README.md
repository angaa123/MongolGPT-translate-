# MongolGPT-translate-
Монгол хэлдээр GPT ашиглахыг хүссэн учир  googletrans ашиглан хийсэн тэст юм .


1. install python

2. <cmd> pip install openai

3. <cmd> pip install googletrans

4. (open gpt.py) Replace API OPENAI_API_KEY      
openai.api_key = "OPENAI_API_KEY"

5. run main.py

6. харилцах

7. гарахдаа "Ctrl" + "C"
8. амжилт хүсий
